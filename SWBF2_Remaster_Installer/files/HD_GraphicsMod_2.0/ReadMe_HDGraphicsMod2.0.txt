Battlefront II 2018 Graphics Mod version 2.0 by HarrisonFog. 


1 - Installation:

Move the contents of the GraphicsMod folder into your gamedata folder.
Turn Anti aliasing off.
Trigger key in game: F11 (buggy, you need to try several times).

2 - Improvements:

Brightness Limiter, sharper and more defined image, lighting FX improvements, corrected saturation, soft vignette.
